# 🌿 Spices Website

An e-commerce web application developed as an academic project for **SLIIT City Uni**. The website provides an online platform for customers to browse spices, manage carts, place orders, and interact with the store through a modern and responsive interface.

## 👨‍💻 Author

**Pasindu Kaumika**  
SLIIT City Uni

---

## 📖 Project Overview

The Spices Website is designed to simulate a real-world online spice store. It focuses on user experience, product browsing, shopping cart management, checkout workflows, and customer interaction features.

## ✨ Features

- Responsive user interface
- Home page with featured content
- Product catalog and product details
- Shopping cart functionality
- Checkout process
- User registration and login
- Order tracking and order history
- Customer reviews and blog section
- Contact page

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Responsive Web Design
